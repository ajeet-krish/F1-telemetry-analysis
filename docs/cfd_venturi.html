CFD Venturi Tunnel

SU2 RANS simulation of the floor venturi tunnel with moving ground wall

Venturi Ground Effect Simulation

The floor of an F1 car acts as a venturi tunnel between the car body and the ground. This simulation uses SU2 CFD to solve the RANS equations for a 2D venturi profile with a moving ground wall, matching the real condition where the track surface moves relative to the car.

$$\frac{\partial \bar{u}_i}{\partial x_i} = 0$$
$$\bar{u}_j \frac{\partial \bar{u}_i}{\partial x_j} = -\frac{1}{\rho} \frac{\partial \bar{p}}{\partial x_i} + \nu \frac{\partial^2 \bar{u}_i}{\partial x_j \partial x_j} - \frac{\partial \overline{u_i' u_j'}}{\partial x_j}$$

Simulation Setup

The 2D venturi domain consists of:

  • Moving bottom wall: Ground plane moving at vehicle speed
  • Profiled top wall: Venturi-shaped floor with diffuser exit
  • Inlet: Uniform velocity inlet (40-80 m/s)
  • Outlet: Pressure outlet at ambient conditions
  • Symmetry: Far-field top boundary

The simulation uses the SU2 RANS solver with the SST k-omega turbulence model. The mesh is a structured quadrilateral grid with refined boundary layers (y+ < 1).

Setup Script

The SU2 configuration and mesh generation are handled by: 

from src.cfd.su2_runner import SU2Config, SU2Solver

config = SU2Config(
    mesh_file="venturi_2d.su2",
    physics="inc_rans",
    reynolds=2.5e6,
    mach=0.1,
    moving_wall=True,
    wall_name="ground",
)
solver = SU2Solver(config)
solver.run()
results = solver.parse_history()
